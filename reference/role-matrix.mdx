Book2befit has three roles: **Owner**, **Admin**, and **Coach**. This page is the definitive reference for what each role can do.

## Dashboard

| Action | Owner | Admin | Coach |
|---|:---:|:---:|:---:|
| View KPI cards | ✓ | ✓ | — |
| View charts & trends | ✓ | ✓ | — |
| View today's sessions | ✓ | ✓ | Own only |
| View upcoming bookings | ✓ | ✓ | Own only |
| Quick-add booking | ✓ | ✓ | — |
| Quick-add client | ✓ | ✓ | — |
