---
title: "Spaces & Equipment"
description: "Manage the rooms, studios, and equipment inside your locations."
---

**Spaces** are the rooms or areas within a location (e.g. Studio A, Spin Room, Treatment Room). **Equipment** is the inventory tracked within those spaces.

## Spaces

### Creating a space

Go to **Spaces** and click **+ Add Space**.

| Field | Description |
|---|---|
| Name | Display name (e.g. "Studio A", "Spin Room") |
| Description | Optional — helps coaches identify the space |
| Capacity | Maximum number of people. Session capacity cannot exceed this. |
| Location | Which location this space belongs to |
| Status | Active or Inactive |

Click **Create Space**. The space is now available to select when creating a session.

### Capacity enforcement

When creating or editing a session, if you select a space with a capacity limit, you cannot set the session capacity higher than the space's capacity. This prevents overbooking at the room level.

### Editing a space

Expand a space row and edit the capacity or other fields inline. Click **Save**. Capacity changes apply to future sessions only — existing sessions are not retroactively adjusted.

### Duplicating a space

Use the **Duplicate** action to quickly create a copy of a space with the same settings. Useful when adding multiple similar rooms.

### Deactivating a space

Set a space's status to **Inactive** to remove it from the session creation form. Existing sessions in that space are not affected.

## Equipment

Equipment tracking lets you keep an inventory of the gear in each space — useful for maintenance scheduling, insurance, and asset tracking.

### Adding equipment

Go to **Equipment** (under the Spaces section) and click **+ Add Equipment**.

| Field | Description |
|---|---|
| Name | Equipment name (e.g. "Reformer 1", "Spin Bike 3") |
| Serial number | Optional — for warranty or insurance records |
| Space | Which space it lives in |
| Status | Available · In Use · Maintenance · Retired |
| Purchase date | Optional |
| Notes | Internal notes (e.g. last service date) |

### Managing equipment status

Update equipment status as needed — e.g. set a bike to **Maintenance** when it's being serviced. This gives your team a live view of what's available in each room.

<Accordion title="Is equipment shown to clients?">
  No — equipment is internal only. Clients never see equipment details on the public booking page.
</Accordion>

<Accordion title="Can I track equipment across multiple locations?">
  Equipment is always linked to a space, and each space belongs to a location. Filter your equipment list by location to see what's in each studio.
</Accordion>
