---
title: "Feedback"
description: "Collect post-session ratings and reviews from clients."
---

After attending a session, clients are invited to leave a star rating (1–5) and an optional written comment. This gives you a direct line to client satisfaction — without needing a third-party review platform.

## How feedback is collected

A feedback request is sent automatically by email after a session ends. The email contains a link to:

```
app.book2befit.com/[slug]/feedback
```

Clients submit their rating and comment on that page. No login is required to leave feedback.

## Viewing feedback

Go to **Feedback** in the dashboard to see all submitted reviews.

Each entry shows:

| Field | Description |
|---|---|
| Client name | Who left the feedback |
| Session name | Which class it was for |
| Session date | When the session took place |
| Coach name | Who led the session |
| Rating | 1–5 stars |
| Comment | Optional written review |
| Submitted at | When the feedback was submitted |

## Filtering feedback

Filter the feedback list by:
- **Rating** — show only 5-star reviews, or identify sessions with low scores
- **Coach** — track satisfaction per team member
- **Activity** — see which class types get the best feedback
- **Date range** — review a specific period

## Using feedback

Feedback data feeds into your analytics — average rating per session type, per coach, and over time are visible in the Analytics dashboard (Growth+ required).

<Tip>
  Use the feedback list regularly during team check-ins. Low scores on a specific coach or class type are an early signal worth acting on.
</Tip>

<Accordion title="Can I respond to feedback?">
  Not directly through the app at this time. If a client leaves contact details and you want to follow up, you can do so manually via the Clients section.
</Accordion>

<Accordion title="Can feedback be made public?">
  Feedback is internal only — it does not appear on the public booking page. It's a tool for your team, not a public review system.
</Accordion>

<Accordion title="Can I disable feedback emails?">
  Feedback request emails are sent automatically. To stop them, contact support. Individual session-level opt-out is not available at this time.
</Accordion>
